# ☕ Midnight Brew.

> **Crafted Coffee. Cozy Moments.**  
> A premium, cinematic, and responsive web experience designed for a modern luxury café brand. Localized for an elegant coffee house aesthetic with seamless dark/light theme switching, smooth transitions, and high-contrast typography.

---

## ✨ Features

* **🎭 Ambience Mode**: Seamlessly transition between **Midnight (Dark)** and **Warm Cream (Light)** modes, built with HSL variables designed to keep reading contrast perfect under any lighting.
* **🎥 Cinematic Hero Experience**: Immersive hero section utilizing soft gradients and optimized overlay balance that lets high-end café visuals shine through while keeping typography highly readable.
* **⚡ Premium Loading Screen**: A custom preloader featuring a realistic steam animation, transitioning smoothly into the main website experience.
* **📜 Interactive Signature Menu**: Beautiful responsive card grid with active categorization filters (Coffee, Main Course, Desserts, Drinks) and pricing styled in Indian Rupees (₹).
* **📅 Soft Glow Reservation CTA**: A refined, elegant floating reservation button designed using a multi-layer soft shadow system that avoids flashy neon effects.
* **💬 Guest Experiences Carousel**: A custom, touch-friendly testimonial slider with active slide magnification and high-fidelity transitions.
* **✨ Warm Glassmorphism Navbar**: Highly legible, responsive navbar featuring a blurred translucent backing (`rgba(239, 235, 229, 0.94)` with `backdrop-filter`) that adjusts dynamically when scrolled.
* **📈 Premium SEO & Performance**: Semantic HTML5 markup, unique selector identifiers, optimized layouts, and modular vanilla code for lightning-fast loads.

---

## 🛠️ Technology Stack

* **Structure**: Semantic HTML5
* **Styling**: Modern CSS3 (CSS Variables, Grid/Flexbox layouts, `@keyframes` animations, and cubic-bezier transition curves)
* **Interactions**: Vanilla JavaScript (ES6 DOM manipulation, Intersection Observer API for scroll animations, and carousel tracking)
* **Icons & Fonts**: Google Fonts (*Cormorant Garamond* & *Outfit*), FontAwesome Icons

## 🚀 Getting Started

No compilers, bundlers, or heavy installations are required. To view and work with the website locally:

### Option 1: Direct File Launch
Simply double-click the `index.html` file or drag it directly into any modern web browser.

### Option 2: Local Web Server (Recommended)
To fully enjoy smooth asset loading and backdrop-filter behavior:
1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension.
3. Click the **Go Live** button at the bottom-right of the window.

## 🎨 Design Guidelines

* **Typography**:
  * Headings use *Cormorant Garamond* (italic variants added for emphasis).
  * Body copy uses *Outfit* for crisp modern readability.
* **Color Palette**:
  * **Midnight (Dark)**: Rich charcoal (`#1A120B`), espresso accents (`#3C2A21`), and champagne cream (`#F5F5F5`).
  * **Warm Cream (Light)**: Soft linen beige (`#F8F5F2`), deep dark chocolate typography (`#2C1E16`), and elegant dark gold (`#B8860B`) highlights.
* **Hover Dynamics**:
  * All hover interactions run on a custom cubic-bezier curve (`cubic-bezier(0.25, 0.8, 0.25, 1)`) to simulate the organic, soft easing characteristic of premium luxury brands.
